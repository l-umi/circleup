1. 회원가입 및 로그인
- 헤더 우측의 회원가입 버튼 클릭
- 정보 기입 (역할은 유저로) 후 회원가입 완료

- 헤더 우측의 로그인 버튼 클릭
- 가입 시 기입한 내용으로 로그인.

2. 강의 검색
- 헤더 좌측의 카테고리로 강의 검색
- 헤더 중앙의 검색창에 "와인" 검색해 "와인 테이스팅" 강의 클릭
- 후기 클릭해 확인 후 수강 신청

3. 강의 페이지 내 서비스
- 페이지 우측에 열린 다시보기 url 클릭해 녹화본 다운로드
- 커리큘럼에 빨간 카메라로 표시된 이번에 열릴 차례인 커리큘럼 클릭
- 실시간강의 접속

4. 실시간 강의
- 이름 변경
- 채팅
- 강퇴 시스템 체크 
- 종료

5. 종료 후 서비스
- 강의 페이지 우측에 최신 다시보기 url 생성 확인
- 후기와 별점 남기기
